J.S. Объекты 9

Сумма свойств объекта
У нас есть объект, в котором хранятся зарплаты нашей команды:
let salaries = {
John: 100,
Ann: 160,
Pete: 130
}

Чтоб запустить задачу нужно обновить страницу!

let salaries = {
John: 100,
Ann: 160,
Pete: 130
};
let sum = 0;
for (let key in salaries) {
sum += salaries[key];
}
alert(sum); // 390

код для суммирования всех зарплат и сохраните
результат в переменной sum. Должно получиться 390.
Если объект salaries пуст, то результат должен быть 0.
