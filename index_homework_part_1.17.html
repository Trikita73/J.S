Сортировать в порядке по убыванию
let arr = [5, 2, 1, -10, 8];
// ... ваш код для сортировки по убыванию
alert( arr ); // 8, 5, 2, 1, -10

Чтоб запустить задачу нужно обновить страницу!

let arr = [5, 2, 1, -10, 8];
arr.sort((a, b) => b - a);
alert( arr );
